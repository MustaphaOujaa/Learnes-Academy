# Guide pratique des composants

Ce document répertorie les principaux éléments d'interface utilisés sur le site.

## Navigation et Structure
*   **Navbar & TopBar** : Gèrent la navigation principale et les annonces importantes en haut de page.
*   **Footer** : Regroupe les liens utiles et les réseaux sociaux.

## Sections d'information
*   **Hero** : Le premier bloc visuel de l'accueil.
*   **Mission & History** : Présentent les valeurs et le parcours de l'école.
*   **AcademicComponents** : Détails sur les méthodes d'apprentissage.

## Inscription et Tarifs
*   **StudentForm** : Le formulaire dynamique pour les nouvelles inscriptions.
*   **FeeStructure** : Les tableaux récapitulatifs des frais de scolarité.
